#### Homebrew (macOS) के माध्यम से

```bash
brew install epicsagas/tap/collet
```

#### cargo-binstall के माध्यम से (सबसे तेज़ - प्री-बिल्ट बाइनरी)

```bash
cargo binstall collet
```

> इसके लिए पहले [`cargo-binstall`](https://github.com/cargo-bins/cargo-binstall) इंस्टॉल होना चाहिए:
> `cargo install cargo-binstall`

#### crates.io के माध्यम से

```bash
cargo install collet
```

#### सोर्स से

```bash
git clone https://github.com/epicsagas/collet.git
cd collet
cargo install --path .
```

## उत्पत्ति और स्थिति

यह प्रोजेक्ट GLM मॉडल्स से अधिक लाभ उठाने के लिए एक व्यक्तिगत खेल के मैदान के रूप में शुरू हुआ था। फिर एक विशेषता से दूसरी विशेषता पैदा हुई — पिछले वाले के ठंडे होने से पहले ही नए मॉडल्स, पेपर्स और प्रतिमान (paradigms) आते रहे।

> "नए मॉडल्स, नए पेपर्स, नए प्रतिमान — किसी भी कमिट लॉग के बने रहने की तुलना में अधिक तेज़।" — एक आवर्ती दुःस्वप्न।

**वर्तमान स्थिति** — अब collet के माध्यम से खुद को ही डॉगफूड कर रहा है, अपनी ही पूंछ खा रहा है।

**स्थिरता**: कोर एजेंट लूप ठोस है। पैरेलल/स्वार्म निष्पादन और बाहरी एकीकरण अभी भी स्थिर हो रहे हैं।
