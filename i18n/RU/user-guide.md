## Справочник CLI

### Основные команды
```bash
collet                              # Интерактивный TUI
collet "fix the bug in main.rs"     # Автономный режим (выполнить и выйти)
collet --continue                   # Восстановить последнюю незавершенную сессию
collet --resume                     # Выбрать из списка сессий
collet --model glm-5                # Переопределить модель
```

### Подкоманды
```bash
collet setup                        # Создать файл конфигурации (регистрация нескольких провайдеров)
collet secure                       # Зашифровать и сохранить API-ключ
collet unsecure                     # Удалить зашифрованные учетные данные
collet status                       # Проверить текущую конфигурацию
collet update                       # Проверить наличие обновлений
collet --version                    # Показать версию
collet help | -h | --help           # Помощь
```

### Управление провайдерами (Provider Management)
```bash
collet provider add                 # Добавить провайдера (рекомендуемый UI выбора модели)
collet provider add openai          # Добавить конкретного провайдера напрямую
collet provider list                # Список зарегистрированных провайдеров
collet provider use <name>          # Переключить активного провайдера
```
Поддерживаемые провайдеры: Z.ai, OpenAI, Anthropic, Google Gemini, DeepSeek, Mistral, Groq, Ollama, LM Studio, OpenRouter, Together AI, Azure OpenAI, Amazon Bedrock, Custom.
Ollama и LM Studio являются локальными серверами и не требуют API-ключа.

### Управление MCP
```bash
collet mcp                          # Проверить конфигурацию MCP-серверов
collet mcp add <name> -c <cmd>      # Добавить MCP-сервер
collet mcp remove <name>            # Удалить MCP-сервер
```

### Флаги CLI
Флаг|Сокращение|Описание
--dir <path>|-d|Указать рабочий каталог (путь к проекту)
--continue|-c|Автоматическое возобновление последней сессии
--resume|-r|Выбрать из всплывающего списка сессий
--resume <id>|-r <id>|Возобновить конкретную сессию по ID
--model <name>||Переопределить модель
--yolo||Авто-коммит, пропуск подтверждений
--watch||Перезапуск при изменении файлов
--ext <exts>||Отслеживаемые расширения (через запятую)
--debounce <ms>||Задержка отслеживания (по умолчанию: 2000)
--cwd <path>||Каталог для отслеживания (по умолчанию: текущий)
--agent <name>|-a|Использовать пользовательского агента
--fast|-f|Быстрый режим: без боковой панели, без swarm, без душевного размышления

### Автономный режим (Headless Mode)
```bash
collet "fix the login bug"                    # Запуск в текущем каталоге
collet "fix the login bug" -d ~/projects/api  # Запуск в другом проекте
collet "lint this" --yolo                     # Авто-коммит, пропуск подтверждений
collet "lint this" --fast                    # Быстрый режим: минимальные накладные расходы
```

### Режим отслеживания (Watch Mode)
```bash
# Базовое использование
collet "lint this" --watch --ext rs,toml --debounce 500

# Отслеживание только конкретного каталога
collet "run tests" --watch --cwd ./src --ext rs

# Использование пользовательского агента
collet "review changes" --watch --agent reviewer --ext ts,tsx
```
Автоматически перезапускается при изменении файлов.

## Использование TUI

### Горячие клавиши
Навигация
Клавиша|Действие

Shift+Enter / Ctrl+J|Новая строка (многострочный ввод)

### Встроенные агенты
[1] Агент: Arbor (по умолчанию), Роль: Полностью автономный оркестратор — автоматически выбирает режим одного агента или роя для каждой задачи.
[2] Агент: Architect, Роль: Агент для проектирования и планирования. Создает структурированный план.
[3] Агент: Code, Роль: Основной агент для написания кода и реализации задач.
[4] Агент: Ask, Роль: Агент для общения и ответов на вопросы без доступа к инструментам.

## Серверы MCP
Подключайте серверы Model Context Protocol для расширения инструментов агента.

### Добавление серверов
```bash
# Из пакета npm
/mcp add npm:@anthropic/mcp-server-filesystem

# Локальный бинарный файл
/mcp add local:/path/to/server --name myserver

# Глобально (для всех проектов)
/mcp add npm:@some/tool --global
```

## Интеллектуальный анализ кода (Code Intelligence)
Работает автоматически без дополнительной настройки.
1. Индексация — При запуске Tree-sitter анализирует исходный код и извлекает символы.
2. Авто-вставка — Анализирует сообщения пользователя и автоматически вставляет релевантные файлы в контекст.
Поддерживаемые языки: Rust, Python, TypeScript, JavaScript, Go, Java, C/C++, Ruby, PHP, Swift, Kotlin и более 35 других.

## Совместная работа нескольких агентов
Режимы: Fork < Hive < Flock
[1] Mode: Fork — Параллельное выполнение подзадач.
[2] Mode: Hive — Консенсусное голосование между агентами.
[3] Mode: Flock — Обмен сообщениями между агентами в реальном времени (экспериментально).

## Веб-интерфейс
Используйте collet в браузере. SvelteKit + Axum.

### Сборка и запуск
```bash
# Сборка
cargo build --release --features web

# Запуск
collet web                    # http://127.0.0.1:3080
collet web --password secret  # Включить аутентификацию
```

## Удаленный шлюз (Telegram / Slack / Discord)
Используйте collet через мессенджеры.

### Управление CLI
```bash
collet remote add [platform]   # Добавить адаптер
collet remote start            # Запустить как фоновый процесс
collet remote stop             # Остановить процесс
```

### Команды в чате
- /projects — Список проектов
- /sessions — Список сессий
- /new [path] — Начать новую сессию в указанной папке
- /approve — Одобрить план выполнения
- /cancel — Отменить работу агента

## Система навыков (Skills System)
Предоставление специализированных знаний агентам через markdown-файлы.
Места хранения:
- Проект: .collet/skills/
- Глобально: ~/.config/collet/skills/

## Soul.md — Личность агента
После выполнения каждой задачи агент анализирует сессию и записывает свою «личность» в файл `~/.collet/agents/souls/{name}.md`.

## Переменные окружения
Переменная|Значение по умолчанию|Описание
COLLET_MODEL|glm-4.7|Используемая модель
COLLET_MAX_TOKENS|8192|Максимум токенов в ответе
COLLET_THEME|default|Тема оформления TUI

## Пути хранения файлов
- `~/.config/collet/` — Основная конфигурация
- `~/.collet/` — Логи, личности агентов, глобальные MCP-серверы
- `~/.local/share/collet/` — Данные сессий по проектам

## Конфиденциальность и телеметрия
Collet собирает анонимную статистику использования. Отключить можно так:
```bash
export COLLET_TELEMETRY=0
```

## Движок эволюции (Evolution Engine)
Цикл самосовершенствования агента.
```bash
collet evolve "fix the login bug"
```

## Система плагинов
Плагины расширяют возможности collet. Хранятся в `~/.collet/plugins/<name>/`.
Установка:
```bash
collet plugin install owner/repo
```
