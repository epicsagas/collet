# <center>collet</center>

<center>
**منسق برمجة وكيلي لا يلين مع دورات وكلاء خالية من الضياع.**

ينسق أي نموذج لغوي كبير (LLM)، وأي وكيل CLI، عبر واجهة المستخدم الطرفية (TUI)، والويب، وبيئة التطوير المتكاملة (IDE).

يقضي على مشكلة التوقف ("stuck") الشائعة في الأدوات القائمة على Node.js من المصدر، مما يوفر ذكاء كود عالي المستوى وتحريراً دقيقاً على أساس أمان Rust.

## الميزات

![الميزات](../../assets/features.jpg)

| الميزة | الوصف |
|---------|-------------|
| **متعدد المزودين** | يتصل بمزودي نماذج لغوية كبيرة مختلفين مع بث SSE |
| **خريطة مستودع Tree-sitter** | استخراج رموز AST للغات Rust/Python/TypeScript/Go + تصنيف PageRank |
| **بحث الكود BM25** | فهرسة تدريجية، محلل رموز واعٍ بالكود، حقن تلقائي للسياق |
| **تشفير BLAKE3 التدريجي** | فحص مسبق لـ mtime + تشفير المحتوى لإعادة تحليل الملفات المتغيرة فقط |
| **تحرير جراحي** | استبدال دقيق للسلاسل النصية دون إعادة كتابة الملف بالكامل |
| **مهارات الوكيل** | نظام مهارات تدريجي مكون من 3 مستويات يعتمد على YAML frontmatter |
| **وكيل فرعي** | تنفيذ متوازٍ للمهام الفرعية في نوافذ سياق معزولة |
| **استمرارية الجلسة** | استعادة الجلسة عبر `--continue` / `--resume` مع حفظ تلقائي |
| **ضغط تلقائي** | ضغط يعتمد على الصلة: إزالة التكرار SimHash + حفظ حرفي مسجل + تراجع إلى ملخص مهيكل |
| **منع التوقف** | حدود التكرار، قواطع الدائرة (circuit breakers)، مهلات Tokio |
| **السمات والرسوم المتحركة** | 6 سمات ألوان (Dracula، Catppuccin، إلخ) + رسوم متحركة للتفكير (Braille spinner) |
| **مراجعة خطة المهندس المعماري** | موافقة المستخدم مطلوبة للانتقال من المهندس المعماري إلى الكود، مع دعم تصدير ملف الخطة |
| **وضع المتوازي/الفريق** | تقسيم المهام ← تنفيذ متوازٍ ← دمج النتائج. وضع الفريق: تواصل بين الوكلاء في الوقت الفعلي + إجماع |
| **LSP/MCP** | أكثر من 35 لغة، أكثر من 48 خادماً مع تراجع تلقائي + تكامل أدوات MCP |
| **بوابة الموافقة على الأدوات** | موافقة على تنفيذ الأدوات من 3 مستويات (Yolo/Auto/Manual)، قابلة للتبديل أثناء التشغيل عبر `Shift+Tab` |
| **بحث مستندات RAG** | مستندات Alcove المحلية + خدمات HTTP Bridge الخارجية. يبحث الوكيل بشكل مستقل عبر أداة `rag_search`؛ تتم مشاركة المعرفة تلقائياً في وضع السرب |
| **Soul.md** | نظام شخصية وكيل مستمر. يسجل الوكيل بنفسه أقسام الهوية والصوت والعالم الداخلي والنمو لبناء شخصية فريدة تدريجياً. تبديل عالمي/لكل وكيل. **مكالمة API إضافية واحدة لكل إكمال مهمة** (بحد أقصى 512 توكن) |
| **تكامل IDE** | تكامل أصلي مع JetBrains + امتداد VSCode عبر خادم ACP |

## البدء السريع

### المتطلبات

- Rust 1.78+ (إصدار 2024)
- مفتاح API لمزود النموذج اللغوي الكبير

### التثبيت والتشغيل

#### عبر Homebrew (macOS)

```bash
brew install epicsagas/tap/collet
```

#### عبر cargo-binstall (الأسرع - ملفات ثنائية مسبقة البناء)

```bash
cargo binstall collet
```

> يتطلب تثبيت [`cargo-binstall`](https://github.com/cargo-bins/cargo-binstall) أولاً:
> `cargo install cargo-binstall`

#### عبر crates.io

```bash
cargo install collet
```

#### من المصدر

```bash
git clone https://github.com/epicsagas/collet.git
cd collet
cargo install --path .
```

### الإعداد والتشغيل
```bash
collet setup # أو استخدم خيار --advanced لإعدادات أكثر تفصيلاً

# واجهة المستخدم الطرفية (TUI)
collet

# وضع Headless
collet "hello collet!"
```
